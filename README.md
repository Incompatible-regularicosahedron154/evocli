It remembers your project history and performs tasks to help you write code faster. It uses standard protocols to connect with other AI tools.

## 🛠️ System requirements

Ensure your computer meets these needs before you start.

* **Operating System:** Windows 10 or Windows 11.
* **Processor:** A modern multi-core processor.
* **Memory:** At least 8 gigabytes of RAM.

## 🚀 Setting up the application

Follow these steps to run the software on your Windows machine.

1. Open your Downloads folder.
2. Double-click the file you downloaded.
3. Windows may show a security window. Click "More info" and then "Run anyway" if the system protects you.
4. A window will open titled "evocli".
5. The application will ask you to enter an API key for your AI provider. You can get this key from the website of your service provider, such as OpenAI or Anthropic.
6. Paste the key into the prompt and press Enter.
7. The application will confirm the connection.

## 💡 How to use the interface

The application uses a full-screen display in your terminal. This view puts all your tools within reach. Use your keyboard to interact with the software.

* **Navigate menus:** Use the arrow keys on your keyboard to move highlights.
* **Select options:** Press the Enter key to open a folder or start a task.
* **Return home:** Press the Escape key to go back one step.
* **Close the app:** Press "Q" on your keyboard to exit.

## 🧠 Using the AI assistant

The core feature of evocli is the AI engine. It acts as a partner for your coding tasks.

1. Open a folder that contains your project files.
2. Press "C" to start a new chat session.
3. Type your request in plain English. For example, you might ask, "Find errors in this script" or "Explain what this file does."
4. The AI analyzes your files and suggests changes or provides answers.
5. If the AI suggests a code change, you see the difference in a split screen. Press "A" to accept the suggested changes or "R" to reject them.

## 💾 Remembering your work

The tool tracks your past interactions. It stores these in a local memory file. This allows the AI to recall your preferences and previous code structures even after you close the program. You do not need to repeat instructions every time you start a new session. The software keeps this data on your hard drive to protect your privacy.

## 🔌 Connecting to other tools

evocli follows the Model Context Protocol. This protocol allows the software to talk to other apps and data sources. If you use other development tools, the software can pull information from them automatically. You can check the "Settings" menu to view active connections. Use the "Add connection" button to link new data sources if your project requires external information.

## 🛠️ Troubleshooting common issues

If you face trouble, follow these steps to fix the issue.

* **App will not start:** Ensure you have an active internet connection. Check that no other terminal window blocks the port.
* **AI returns an error:** Verify your API key in the settings menu. Ensure your account with the AI provider has enough credit.
* **Slow performance:** Close other heavy programs on your computer. Large projects use more memory, so consider closing other browser tabs.
* **Interface looks messy:** Resize your terminal window. The interface adjusts to the width of your screen. 
* **Key shortcuts do not work:** Click inside the terminal window to ensure it is the active window.
